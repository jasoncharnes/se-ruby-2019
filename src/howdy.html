Afternoon Workshops (1:30 pm - 4:30 pm)

  • Build Your Own Web Framework to Understand Rails Magic
    • Room: Letterpress
    • Please have any recent version of Ruby with RubyGems and Bundler installed
    • Also Git
    • For extra credit, install the Rack gem 2.0.7 or higher - otherwise we will during the workshop
  • Dockerizing Rails - A supercharged development process
    • Room: Woodblock
    • During the Dockerizing Rails' workshop, we will download several GB of data. This might be tricky with the WiFi on site. The good news is, you can pre-download all the prerequisites!
    • Please head to Github and follow the `Preparing for the workshop` section. You will install Docker, test the setup and also download some "images" that we will use during the workshop. Following the instructions should reduce the data requirements to just a couple MB and speed everything up a lot.
